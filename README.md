# OpenCV Playground

### This is a small project where you can experiment with image processing using OpenCV.

To get started:

1. Download the images mentioned in the notebook.
2. Zip them and upload the archive to your Google Drive.
3. Mount Google Drive in Google Colab and adjust the path if needed.
4. Make sure the image file names match those used in the notebook.
5. Click **Runtime => Run all** and enjoy!
